INSTALLAZIONE PLUGIN LITTLEFS PER ARDUINO IDE 1.8.x (ESP32/ESP32-S3)
=====================================================================

REQUISITI:
- Arduino IDE 1.8.x (NON funziona con Arduino IDE 2.x)
- Core ESP32 installato (versione 2.0 o superiore)

ISTRUZIONI:
1. Chiudi Arduino IDE se aperto

2. Vai nella cartella Arduino (tipicamente):
   C:\Users\TUONOME\Documents\Arduino

3. Crea la cartella "tools" se non esiste:
   C:\Users\TUONOME\Documents\Arduino\tools

4. Copia la cartella "ESP32FS" (inclusa in questo pacchetto) dentro "tools"

   Struttura finale:
   Arduino\
     tools\
       ESP32FS\
         tool\
           esp32fs.jar

5. Riavvia Arduino IDE

6. Vai in Tools - dovresti vedere "ESP32 Sketch Data Upload"

USO:
1. Crea una cartella "data" dentro la cartella del tuo sketch
2. Metti i file da caricare nella cartella "data"
3. Seleziona la scheda corretta (es. ESP32-S3)
4. Clicca su Tools > ESP32 Sketch Data Upload

NOTE:
- Per ESP32-S3: assicurati di avere il core ESP32 versione 2.0+
- Seleziona un Partition Scheme che includa spazio per SPIFFS/LittleFS

Plugin scaricato da: https://github.com/lorol/arduino-esp32fs-plugin
